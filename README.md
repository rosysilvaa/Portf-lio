# 💻 Meu Portfólio Pessoal


## 🚀 Sobre o Projeto

O objetivo deste projeto é centralizar minhas experiências, projetos e formação acadêmica em uma interface limpa, moderna e responsiva. Ele reflete meu momento atual de retorno focado ao mercado de desenvolvimento web.

---

## 🛠️ Tecnologias Utilizadas

Para a construção deste portfólio, utilizei as tecnologias fundamentais da web:

- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Estilização moderna, layouts responsivos (Flexbox/Grid) e animações sutis.
- **JavaScript (ES6+):** Manipulação de DOM e interatividade.
